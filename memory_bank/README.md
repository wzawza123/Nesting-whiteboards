## 项目特色

### 🎯 核心亮点
- **无限嵌套**: 每个矩形节点都是一个子图，支持无限层级嵌套
- **多种节点**: 矩形、文本、图片三种节点类型，各具特色
- **现代化**: 基于最新的Web技术栈，支持现代浏览器特性

### 🛠️ 技术架构
- **AntV G6**: 强大的图形可视化引擎
- **模块化设计**: 清晰的模块分工和依赖关系
- **事件驱动**: 完善的事件系统支持复杂交互

### 🎨 用户体验
- **直观操作**: 符合直觉的交互设计
- **快捷键**: 全面的键盘快捷键支持
- **实时反馈**: 丰富的视觉反馈和状态提示

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```
